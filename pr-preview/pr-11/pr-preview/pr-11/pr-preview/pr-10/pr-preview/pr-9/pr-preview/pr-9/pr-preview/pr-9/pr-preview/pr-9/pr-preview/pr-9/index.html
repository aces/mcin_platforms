The McGill Centre for Integrative Neuroscience (MCIN), led by Dr. Alan Evans at McGill University, creates open neuroinformatics platforms to conduct computationally-intensive brain research using innovative mathematical and statistical approaches to integrate clinical, psychological or neuroimaging phenotypes with genotypic information. MCIN constitutes the neuroinformatics component of the the Ludmer Centre for Neuroinformatics and Mental Health and coordinates global collaborative initiatives including the Canadian Open Neuroscience Platform (CONP)
